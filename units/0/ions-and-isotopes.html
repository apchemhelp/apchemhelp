Ions and Isotopes

Ions

Ions form when the number of protons in an atom is not the same as the number of electrons in an atom. This usually happens because an atom has gained or lost electrons and normally does not occur as a result of a change in protons. 

For example, if we take the element Oxygen, which has an atomic number of 8, and says that it has a charge of negative two, what we mean is that in the oxygen atom there are 8 protons, but 10 electrons. Since electrons have a negative charge when there are more electrons than protons that means that the charge is negative. 

Charge of Ion = number of protons - number of electrons.

If the charge of an ion is positive, then that means that there are more protons in an atom than there are electrons. 

 

Isotopes

The number of protons in an atom stays constant, and the number of electrons in an atom can change, but what about neutrons? Different atoms of the same element can have different numbers of neutrons. These are called isotopes. Not all isotopes are stable and many are unstable. There are many ways that you can write an isotope such as Oxygen-16 where oxygen is the element and 16 is the mass number. Another isotope of Oxygen would be Oxygen-17, which means that it has one more neutron than the other isotope.

 In order to find the number of protons in an atom what you can do is subtract the atomic number from the mass number.
