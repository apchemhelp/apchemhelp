Periodic Table of Elements

Once upon your lifetime, you probably saw this table before (to view the “AP” version of the periodic table of the element, check the resource tab). 

This is the periodic table of elements, the arrangement of different elements in our universe. This table contains pretty much the most basic properties for every element that we have discovered so far. Let’s dig deeper into the periodic table. 

Let’s take a look at a specific element, or hydrogen, as an example. 

The top number represents the atomic number, or how many protons does an element have. This defines this specific element. In other words, each element has a different number of protons. 

The bolded letter is the symbol for the element. This is how scientists refer to different elements. In this case, H represents Hydrogen.

Finally, the decimal is the mass number. The mass number represents how “heavy” an element is by adding the mass of each proton, neutron, and electron (electron’s mass is so small that it usually isn't considered as part of the mass number). 

Now let’s look at a bigger picture rather than a single element. Just like every other table around the world, the periodic table has columns and rows. Columns are called families, or groups, in the periodic table; each family shares similar properties among each other. Rows are called periods in the periodic table.
